# To-do-List
A clean, lightweight to-do list built with HTML, CSS, and vanilla JavaScript. Designed for simplicity and speed, it helps users capture tasks instantly, mark them complete, and persist progress using localStorage—no backend, no friction.

Key highlights

Add, complete, and delete tasks effortlessly

Persistent data with browser localStorage

Minimal, modern UI with custom styling

Responsive layout using Bootstrap

Zero dependencies beyond the browser

This project focuses on clarity over complexity—proof that powerful user experiences don’t need heavy frameworks. Perfect as a beginner-friendly project or a foundation to evolve into a smarter productivity tool.
